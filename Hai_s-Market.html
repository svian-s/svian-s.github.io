Hai's Market is a portfolio project by Asvian Sulaeman, showcasing a sample marketplace app developed with Flutter. The app follows the MVVM architectural pattern, utilizing key technologies like dependency injection, REST APIs, and the Hive database for local storage. It features integration with two fake APIs: Dummy JSON (https://dummyjson.com/) for product data and the Platzi Fake API (https://fakeapi.platzi.com/) for user authentication. This project demonstrates a full-stack implementation, highlighting Asvian's skills in modern app development practices.
