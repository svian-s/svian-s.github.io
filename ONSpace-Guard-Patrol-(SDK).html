ONSPACE GUARD PATROL

Guard patrol is an SDK to track security guards movements while on patrol duty in onspace, patrol result data will be visible on the web portal. It's using Android phone with foreground & background service to actively detect nearby BLE terminal which mapped in the server. We sync the data every time period on custom-able setting (e.g: 10 minutes). This SDK integrated to OCBC internal app for security.
